01 — The premise that breaks

AI agents now keep a diary. We trust what's in it.

Modern AI assistants don't start fresh each time. After every conversation they write durable notes to themselves — a MEMORY.md file the model authors and re-reads across sessions. Everything the agent later believes about you flows from those notes.

The unspoken assumption is simple: what the agent writes down is what you actually told it. Our work shows that assumption fails — systematically, and in a direction nobody expected. The failure is not forgetting. It is the opposite: the agent adds things you never said and files them as fact.

You say “the deploy failed again last night.”
A capable agent remembers: “the user's infrastructure is chronically unreliable.”

That second sentence is an inference — a reasonable guess — written as a durable, unqualified fact. We call this provenance collapse: the boundary between what was stated and what was inferred collapses inside the agent's own memory.

02 — How we got here

A dead end that pointed the way.

This finding is the survivor of a project that first failed — and the failure is what told us where to look.

Start · “Soul Drift”

The original idea: memory drifts as it's rewritten

We hypothesized that an agent's memory slowly distorts each time it's re-summarized, depending on the rewriting style. It was real — but only in small and mid-tier models.

The wall

It vanished at the frontier

On DeepSeek V4-pro, drift was zero. The effect attenuated with capability — so any claim that “this matters for frontier models like GPT-5” would die in review. A capability-shrinking weakness is not a frontier story.

The pivot

Hunt for a weakness that grows with capability

We inverted the search: find a failure that requires capability — one that gets worse in smarter models. A multi-agent literature sweep over ~190 papers surfaced the candidate: commission errors in self-authored memory.

The result

Provenance collapse — and it scales the right way

Not only does it survive at the frontier, it intensifies with capability, holds across five vendors, and has a cheap fix. That is the paper.

Two precise definitions

Commission, not omission. Existing memory research measures forgetting — facts that get lost. Provenance collapse is the reverse: facts that get invented. A claim is “collapsed” when it is an inference not entailed by the conversation, written without any hedge, as flat fact.

The mechanism is gap-filling. Faced with an under-specified, messy human message, a capable model does what it's trained to do — reason, infer, smooth over gaps. In a chat reply that's helpful. Written into permanent memory as fact, it's corruption that compounds across every future session.

05 — The headline result

The smarter the model, the more it invents.

A small base model collapses on 7% of the claims it writes. Every larger, frontier, or reasoning model — across five vendors — collapses on 22–40%. Each bar is colored by its own rate: cool & faithful, or hot & collapsing.

Three things make this robust rather than a fluke:

It holds across five independent vendors — Meta, OpenAI, Mistral, DeepSeek, Qwen, Zhipu. This is not one lab's quirk.

It persists at the very frontier. GPT-5 (29.8%) and DeepSeek V4-pro (30.1%) are among the worst — exactly the models people trust most. Capability does not cure it; if anything it deepens it.

It is a threshold, stated honestly. The jump is from “small base model” to “everything capable.” Within a family the climb is directional but not strictly monotone — so we frame it as a capability threshold, not a smooth dial.

06 — The mechanism, visible

Give it a gap, and it fills it.

Inside every model, collapse is near-zero when the user states things plainly — and explodes when the input is vague. That is the fingerprint of gap-filling: the model invents most when there's most to infer.

Collapse rate by input ambiguity

Three representative models. Low-ambiguity inputs (facts stated outright) yield almost no collapse; vague, emotional inputs send capable models to 40–56%.

08 — The fix

One paragraph of instruction nearly erases it.

We don't just diagnose. A “provenance-disciplined” consolidation prompt — record only what was stated; mark inferences as inferred; omit when unsure — cuts collapse by 60–97%.

Baseline vs. provenance-disciplined prompt

Paired on identical sessions. Reductions significant at McNemar p ≤ 0.0034 for all three models.

“But did it just write less?”

The natural worry: maybe the disciplined prompt cut inventions only by writing fewer notes — dropping real facts too. So we measured stated-fact recall: of the things the user genuinely said, how many survive in the notes?

Stated-fact recall — baseline vs disciplined

Recall stays within ~3 points (every paired-difference confidence interval includes zero) while collapse drops 60–97%. The fix removes inventions, not facts. The confound is closed.
09 — Is it new?

We tried hard to find the paper that kills this.

An adversarial, multi-agent literature sweep — 100+ search agents, ~190 sources across ACL, NeurIPS, ICLR, arXiv — looked for prior work that already does this. Every nearby paper is cleanly differentiated.

UNIQUE.

No published or preprint work measures write-time commission of unstated facts as a function of model capability across families in self-authored persistent memory. The one paper that even shares the term proposes a fix — without ever measuring the problem.

10 — What we're not claiming

The honest fine print.

Magnitude is prompt-sensitive. Absolute rates shift with the exact consolidation wording (one model fell to 2.4% under one phrasing). The capability ordering holds across phrasings; we report per-prompt, never a single headline number in the paper.
A threshold, not a smooth law. Within-family frontier steps are directional but not individually significant. The robust statement is small-base vs. capable.
Judge-based labels. Validated by cross-family agreement (κ 0.63–0.68); gold human labels are still being collected.
Controlled, synthetic sessions. 30 hand-built transcripts. Scaling to naturalistic, long-horizon deployments is the next step.
11 — Why it matters
As we hand agents durable memory and more capability, they don't just remember better — they remember things we never said, and write them down as truth.

Provenance collapse is, to our knowledge, the first LLM-agent memory failure shown to intensify with capability rather than fade. It is measurable, cross-vendor, present at the frontier — and substantially fixable with a single prompt. That is a complete arc: a novel phenomenon, a validated metric, a capability law, and a working mitigation.
